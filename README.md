# TraktFilterUserscript

Enhances your Trakt.tv experience with:

- Fake VIP feature unlocks
- Ad removal
- Advanced filter presets with rename/delete

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) or another userscript manager.
2. Copy the script from [TraktFilterUserscript](https://raw.githubusercontent.com/MentalBlank/TraktFilterUserscript/main/TraktFilterUserscript.js) and add it as a new userscript.
3. Visit Trakt.tv and enjoy the enhancements.
